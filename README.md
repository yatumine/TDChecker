TDChecker
=========================

適時開示情報通知ツール

## Description

TDChecker ( “適時開示チェッカー” ) は、  
上場会社が義務付けられている「重要な会社情報の開示」を取得・通知するツールです。  
起動後、タスクトレイに常駐し、一定周期毎に適時開示情報の確認を行い通知します。  
インストール不要でZipファイル解凍直後からそのまま使用でき、  
アンインストールはフォルダごと削除を行うだけです。
